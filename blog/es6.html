# ECMAScript 6

ECMAScript 6 就是俗称的 es6。本文主要是重看 es6 的时候一些细节或者小坑记录吧。本文大部分内容都是来自阮老师的文章

# 首先搞清楚一个问题 ECMAScript 和 JavaScript 的关系？

1996 年 11 月，JavaScript 的创造者 Netscape 公司，决定将 JavaScript 提交给标准化组织 ECMA，希望这种语言能够成为国际标准。次年，ECMA 发布 262 号标准文件（ECMA-262）的第一版，规定了浏览器脚本语言的标准，并将这种语言称为 ECMAScript，这个版本就是 1.0 版。

因此，ECMAScript 是 JavaScript 规范。

# 第二个问题是 是什么 ES6? 为什么叫 ES6？

ES6 的第一个版本，是在 2015 年 6 月发布了，正式名称就是《ECMAScript 2015 标准》（简称 ES2015）。2017 年 6 月发布 ES2017 标准。 不过现在 ES6 已经是泛指了，含义是 5.1 版以后的 JavaScript 的下一代标准，涵盖了 ES2015、ES2016、ES2017 等等。
